# erp_maker
Ce système est conçu pour la création des ERP dynamiques sans avoir à coder les côtés back. Il consiste à entrer dans l'inteface utilisateur toutes les logiques métiers ainsi que toutes les fonctionnalités attendus par le ERP creator.
Le système erp_maker utilise des technologies backend :

<li><strong>MySQL</strong></li>
<li><strong>Prisma</strong></li>
<li><strong>Node Express Typescript</strong></li>
